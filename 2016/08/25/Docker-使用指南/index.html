Docker 使用指南

Posted on 2016-08-25 Edited on 2016-08-26 In 程序

Docker用作生产环境搭建非常方便

镜像（image）

搜索镜像

docker search + 镜像名称

查看本地镜像

docker images

下载镜像

docker pull + 镜像名称

容器（container）

运行容器

docker run [–name] +昵称 + 镜像名称 + 镜像下的命令

查看正在运行的容器

docker ps

查看运行过的所有容器

docker ps -a

删除容器

docker rm + 容器的id或名字

查看容器

docker logs + 容器名字
docker logs –follow + 容器ID和名字

启动容器

docker start + 容器名字

停止容器

docker stop + 停止容器的名字

重新启动容器

docker restart + 容器名字

交互容器

docker run –interactive -tty centos /bin/bash

后台运行容器

docker run –detach centos ping www.google.com

decorator详解

Posted on 2016-09-04 Edited on 2016-11-07

之前在《Python基础》提过一下decorator, 这次深入探讨和研究一下

  • decorator本质上就是一个高阶函数，他接受一个函数作为参数，返回一个新函数

  • python中decorator用**@** 语法，可以避免使用手动编写f=decorate()

  • 写一个日志装饰器
    def log(f):
    def fn(x):
    print(“call”+f.name+”()…”)
    return f(x)
    return fn

  • 应用日志装饰器
    @log
    def factorial(n):
    return reduce(lambda x,y:x*y,range(1,n+1))
    print(factorial(10))

  • 运行结果
    call factorial()…
    3628800

  • 注意： fn的参数个数，要和之后f的参数个数相同，如果保证任意参数正常调用，使用*args 和**kw
    def log(f):
    def fn(*args,**kw):
    print(“call”+f.name+”()…”)
    return f(*args,**kw)
    return fn

  • 练习：打印函数的调用时间
    def performance(f):
    def fn(*args,**kw):
    print(“call “+f.name+”()…at”+time.asctime())
    return f(*args,**kw)
    return fn

    @ performance
    def addTwo(x,y):
    return x+y

    print(addTwo(1,4))
    call addTwo()…atSun Sep 4 14:20:19 2016
    7

  • 带参数的decorator

    • 原理很简单，就是在标准的decorator外面再包裹一个函数，最后返回decorator本身

  • 练习：带参数的decorator
    import time
    def performance(unit):
    def decorator(f):
    def fn(*args,**kw):
    start = time.clock()
    if unit==”ms”:
    print(“call %s using %f ms”%(f.name,(time.clock()-start)*1000))
    elif unit==”s”:
    print(“call %s using %f s”%(f.name,time.clock()-start))
    else:
    print(unit +”is a unavailable unit”)
    return f(*args,**kw)
    return fn
    return decorator

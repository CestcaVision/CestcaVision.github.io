重温Javascript

Posted on 2016-09-12 Edited on 2016-09-20 In 程序

之前JS没学好，现在重新捋一遍

ES6新功能

变量

let

  • 声明块级作用域的变量
  • var 声明，在for循环等语句块中，无法定义具有局部作用域的变量
    function foo(){
    for (var i=0;i<100;i++){
    }
    i += 100 // 仍然可以引用变量i
    }
  • let声明，就可以创造出真正的块级作用域
    function foo(){
    for (let i=0;i<100;i++){
    sum += i;
    }
    i+=1; // SyntaxError
    }

const

  • 不能定义两次const变量, 如果变量是数组，可以更改变量的内容，但仍不可以重新定义

解构

解构数组

function breakfast(){
    return ["cake","tea","apple"]
}
let [dessert,drink,fruit]=breakfast();
console.log(dessert,drink,fruit);

解构对象

function breakfast(){
    return {dessert:"cake",drink:"tea",fruit:"apple"}
}
let {dessert: dessert,drink: drink,fruit: fruit}=breakfast();
console.log(dessert, drink,fruit);

模板字符串

  • 字符串前后用``括起来
  • ${var}
  • 还可以把字符串用回车切分成多行，不用\n

标签模板字符串

模板字符串的三个判断

  • startsWith()
  • endsWith()
  • includes()

函数

默认参数

function breakfast(dessert="cake",drink="coffee"){
    return \`${dessert} ${drink}\`
}

展开操作符…

  • …数组 可以将数组展开

剩余操作符…

function breakfast(dessert,drink,...foods){
    console.log(dessert,drink,foods);
}
  • 函数除了接受dessert,drink参数外，剩余的参数被当做叫做foods的数组

解构参数

function breakfast(dessert,drink,{location,restaurant}){
    console.log(dessert,drink,location,restaurant)
}
breakfast("cake","tea",{location:"LA",restaurant:"Bubba Gump"})

函数的name属性

  • 函数名称.name 可以得到函数的名字

箭头函数

let sumUp = (x,y)=> x+y;

  • 如果不只是返回数据，函数体外要加上{}
  • 如果函数体没有参数，要加上空白括号

对象

对象表达式

let dessert = "cake";
let drink = "coffee";
let breakfast = {
    dessert,
    drink,
    breakfast()
}
// 等同于对象
{
    breakfast : function(),
    dessert: "cake",
    drink: "coffee"
}

对象属性名

  • 可以用.来为对象添加属性
  • 如果属性包含空格，则必须用[]来添加属性，比如

    food[“hot drink”] = “coffee”

  • 还可以在[]内嵌入变量

Object.is()方法

  • 这个方法用于对比两个值是否相等
  • 因为相等可以用==和===表示，Object.is()主要解决一些JS的语法BUG， 比如

Object.is(NaN,NaN)
true
Object.is(+0,-0)
false

Object.assign()方法

  • 把一个对象的内容赋予另一个对象
  • 语法： Object.assign(终点，源)

Object.setPrototypeOf()方法

  • 创建对象以后改变对象的原型
  • 例子：
    let breakfast = {
    getDrink(){
    return ‘tea’;
    }
    };
    let supper = {
    getDrink(){
    return ‘beer’;
    }
    };
    let sunday = Object.create(breakfast);
    sunday.getDrink();
    // 结果为 “tea”
    Object.setPrototypeOf(sunday, supper);
    sunday.getDrink();
    // 结果为 “beer”

__proto__

  • 可以直接为对象设定原型
  • 设定方法：
    let sunday = {
    proto: breakfast
    };
    sunday.proto = dinner;

super

  • 得到原型对象的属性和方法
    let sunday = {
    proto: breakfast,
    getDrink(){
    return super.getDrink()+”and sunday champagne”;
    }
    }
    console.log(sunday.getDrink())
    // 返回结果 tea and sunday champagne

生成器与迭代器

理解迭代器，自己写一个

function chef(foods){
    let i = 0;
    return {
        next() {
            let done = (i>=foods.length);
            let value = !done ?foods[i++]:undefined
            return {
                value: value,
                done: done
            }
        }
    }
}

生成器

  • function* 就声明了一个迭代器

  • yield 后面加上每次调用yield希望返回的结果

  • 举例：
    function* chef(){
    yield “tomato”;
    yield “egg”;
    }
    let leo = chef();
    leo.next();

  • 另一种用法
    let chef = function* (foods){
    for(var i=0; i<foods.length;i++){
    yield foods[i];
    }
    }
    let leo = chef([“tomato”,”egg”])
    leo.next()

类

  • class
  • 初始化一个类
    class Chef {
    constructor(food){
    this.food = food;
    }
    cook(){
    console.log(this.food);
    }
    }

get and set

class Chef {
    constructor(food){
        this.food = food;
        this.dish = [];
    }
    get bill(){
        return this.dish;
    }
    set menu(piece){
        this.dish.push(piece);
    }
    cook(){
        console.log(this.food);
    }
}
let leo = Chef();
//把每道菜插入到菜单当中
leo.menu = "西红柿炒鸡蛋";
leo.menu = "宫保鸡丁";
// 生成账单
leo.bill

静态方法

  • 可以让类里的方法，不需要实例化就可以使用
  • 关键字 static
  • 举例
    class Chef {
    constructor(food){
    this.food = food};
    static cook(special_dish){
    console.log(special_dish)
    }
    }
    Chef.cook(“火爆腰花”)

继承

  • 让一个类继承另一个类的属性和方法
  • 用super 调用父类的属性
  • 举例
    class Person {
    constructor(name,birthday){
    this.name = name;
    this.birthday = birthday;
    }
    intro() {
    return 我叫${this.name},我出生于${this.birthday}
    }
    }

    class Chef extends Person {
    constructor(name,birthday){
    super(name,birthday);
    }
    }
    let leo = new Chef(“leo”,”1992-3-10”)

集合

Set

  • 类似数组，但不允许有重复的元素
  • 新建Set new Set()
  • .add(element) 添加元素
  • .size 查看元素个数
  • .has(element) 判断是否有某个元素
  • .delete(element) 删除某个元素
  • .forEach(函数) 遍历集合中的每个元素，执行函数
  • .clear() 清空集合

Map

  • Map 是对于Object的补充，Object的键类型有限，但Map就没有这些限制
  • 新建 new Map()
  • .set(key:value) key可以是对象、函数、字符串。。。
  • .size 查看Map 键值对个数
  • .get(key) 得到对应的值
  • .delete(key); 删除
  • .has(key) 判断
  • .forEach((value,key)=>{})
  • .clear() 清空

forEach()方法

  • ES6 新增处理集合、数组、Map的方法, 内嵌函数，遍历处理集合数组中的元素

  • 注意参数的顺序和含义

  • 数组
    var arr = [‘a’,’b’,’c’];
    arr.forEach((element,index,array)=>{
    console.log(element);
    console.log(index);
    console.log(arry)
    })

  • 集合
    var s = new Set([‘a’,’b’,’c’]);
    s.forEach((element,sameElement,set)=>{})

  • Map
    var m = new Map([[1,’a’],[2,’b’],[3,’c’]]);
    m.forEach((value,key,map)=>{})

Module

  • 导出：在js的底部export 变量或者函数
  • 导入：
    • import {导出对象} from “路径”
    • import * as 替换对象 from “路径”

重命名

  • export { dinner as supper };
  • import { breakfast as shit};

默认导出导入

export default + 默认导出的变量或函数
另一种写法 export {变量或函数 as default}
import 随便去取个名字（默认没有大括号）from ‘路径’

函数的补充

arguments关键字

  • 指向当前函数调用者传入的所有参数
    function foo(x) {
    alert(x); // 10
    for (var i=0; i<arguments.length; i++) {
    alert(arguments[i]); // 10, 20, 30
    }
    }
    foo(10, 20, 30);

  • 包括定义时，没有参数，调用时有参数的情况
    function abs() {
    if (arguments.length === 0) {
    return 0;
    }
    for(var i=0;i<arguments.length;i++){
    var x = arguments[i];
    console.log(x>=0?x:-x)
    }
    }
    abs(-9,10); //9,10

  • arguments最常用于判断传入参数的个数，用arguments.length

  • 剩余参数用function (x,y,…rest){} 处理，注意rest不是关键字，可以替换成其他变量

this

  • this 在函数内部，一定值得是调用该函数的对象

  • 如果函数内部嵌套另一个函数，想要在内层函数里引用this，就要在调用内层函数时把this捕获过来。

  • 函数中的this还可以直接指定，用apply方法，apply有两个参数，第一个参数指定函数指向的对象，第二个参数是Array，表示函数本身的参数
    function getAge(){
    var y = new Date().getFullYear();
    return y - this.birth
    } // 这是this指向window

    var xiaoming = {
    name : “小明”,
    birth: 1990,
    age: getAge
    };
    xiaoming.age(); //26
    getAge.apply(xiaoming,[]);//26, this指向xiaoming

  • call 和 apply 的不同

    • call 也可以指定函数中this的指向，
    • call()把参数按顺序传入
    • apply() 把参数打包成数组传输

标准对象

Date

  • 创建 var time = new Date()
  • Date实例几个方法
    • .getFullYear()
    • .getMonth() // 尤其注意JS里的月份是从0开始算起的
    • .getDate()
    • .getDay() // 周几
    • .getHours()
    • .getMinutes()
    • .getSeconds()
    • .getMilliseconds()
    • .getTime() // 实例时间到1970年1月1日0时整GMT时区的毫秒数
  • Date 还可以自定义时间戳 var time = new Date(年,月,日,时,分,秒)
  • 另一种自定义时间戳的方法 var time = new Date(1435146562875)
  • Date的集中自带方法
    • Date.parse(‘2015-06-24T19:49:22.875+08:00’) 可以将ISO 8601格式的字符串 转换成 数字时间戳, 生成 1435146562875
    • Date.now() 可以生成当前调用方法时 到1970年1月1日0时整GMT时区的毫秒数

正则表达式

  • 创建正则表达式

    var rel1 = /^\d{3}-\d{3,8}$/
    var rel2 = new RegExp(‘^\d{3}\-\d{3,8}$’)
    注意，如果使用第二种写法，因为字符串的转义问题，字符串的两个\实际上是一个\

  • 验证字符串用test()方法，比如reg1.test("010-12345")

正则表达式在JS中的应用

切分字符串

‘a,b;; c d’.split(/[\s,;]+/); // [‘a’, ‘b’, ‘c’, ‘d’];

分组

  • 用括号对正则表达式的匹配内容进行分组，用.exec()方法匹配成功后，返回一个Array, 第一个元素是匹配到的整个字符串，后面的元素是括号里匹配到的字符串

JSON

  • JS用来传输数据的方法
  • 序列化：可以把对象转化成JSON串
    • JSON.stringify(对象，[筛选对象的键值]，’ ‘)
    • 第二个参数还可以传入一个函数，对每个键值对进行先处理，函数的第一个参数是键，第二个参数是值
  • 反序列化：把JSON串转换成相应的对象
    • JSON.parse(字符串，function(key,value){})
